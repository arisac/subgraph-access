# Install

`yarn`

### Config network for subgraph.yaml

`yarn generate NETWORK START_BLOCK`

will generate `./subgraph.yaml` using input

- if NETWORK is number, add quotes mannully in generated subgraph.yaml, e.g. `network: 1` to `network: "1"`

### Build All

`yarn build:all`
